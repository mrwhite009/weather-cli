# Contributing to weather-cli

## Setup

```bash
git clone https://github.com/mrwhite009/weather-cli
cd weather-cli
pip install -e ".[dev]"
```

## Running tests

```bash
pip install pytest
python -m pytest tests/ -v
```

## Adding features

1. Open an issue first
2. Add tests
3. Update CHANGELOG
4. Submit a PR

## Code style

Follow PEP 8. Use type hints.
